## Termos que NÃO traduzir

- Nomes próprios: "Premier Ultra", "Premier"
- Marcas: "Supabase", "WhatsApp", "Telegram", "Google Play"
- Jargão do domínio mantido em inglês: "Greens", "Live" (se for nome do produto), "Tip"
- Chaves técnicas (slugs, enums, values internos): `'free'`, `'basic'`, `'pro'`, `'ultra'`, `'green'`, `'red'`, `'pending'`, `'upgrade_basic'`, etc.
- IDs, classes CSS, nomes de variáveis, nomes de componentes React

## Regras gerais

1. **Caracteres especiais:** sempre usar acentuação e `¿`/`¡` no espanhol (`¿Cuál?`, `¡Hola!`).
2. **Tom:** informal, segunda pessoa do singular ("tú/tu") — Chile usa "tú" em produtos digitais (não "vos" nem "usted" formal).
3. **Não traduzir** atributos técnicos (`className`, `id`, `data-*`, `value` de enums).
4. **Comentários de código:** podem ficar em português (são internos, não vão pra UI).
5. **Caso de dúvida:** registrar aqui antes de inventar tradução.
